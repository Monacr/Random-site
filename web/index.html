Pseudorandom number generators suck, but we can still use them to get 'random number' sequences, or rather approximately random number sequences. Shown we can see visualization examples of the approximate random uniform distribution method, Middle Square method, Xorshift method, and Linear Congruential method of pseudo-random number generation.
